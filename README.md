# Docker-compose-base

## Enter the port you want to run

```
export PY_PORT=5000
```

## build the App

```
$ docker-compose up
```

1. Enter http://0.0.0.0:5000/ in a browser to see the application running.

You should see a message in your browser saying:

```
Toronto
```
